## Плюсы

Рассказать о плюсах Wuki UI в сравнении с другими UI библиотеками

- Ноль Dev зависимостей
- Самый маленький размер
- Кастомизация
- Никто не думает об удобстве разработчиков

```
<Space size={3} horizontal />
```

Этот фрагмент кода взят из другой UI библиотеки. Зачем использовать 2 параметра, если можно сократить это до одного, тем самым упростив жизнь разработчику и уменьшить код:

```
<Space x={3} />
```
